# Анализ лояльности и удержания пользователей Яндекс Афиши

## Описание проекта
Аналитическое исследование поведения пользователей сервиса Яндекс Афиши с целью выявления ключевых факторов, влияющих на возвратность клиентов (retention). На основе исторических данных о покупках за 2024 год построены пользовательские профили, проверены продуктовые гипотезы и сформированы дата-ориентированные рекомендации для маркетинга.

## Цель и задачи
- Выгрузить и подготовить данные из PostgreSQL с использованием SQL-запросов и оконных функций.
- Сформировать агрегированные профили пользователей (витрина данных).
- Провести EDA и выявить корреляции между признаками первого заказа и вероятностью повторной покупки.
- Проверить продуктовые гипотезы о влиянии типа мероприятия, региона, дня недели и среднего чека на лояльность.
- Предоставить actionable-рекомендации по удержанию аудитории.

## Стек технологий
- **Язык:** Python 3.10+
- **Анализ данных:** `pandas`, `NumPy`, `phik` (нелинейные корреляции)
- **Визуализация:** `Matplotlib`, `Seaborn`
- **Работа с БД:** `SQLAlchemy`, `PostgreSQL` (CTE, `JOIN`, оконные функции `LAG/OVER`)
- **Конфигурация:** `python-dotenv`
- **Среда:** Jupyter Notebook

## Данные и предобработка
- **Исходный объем:** ~290 611 транзакций из БД Яндекс Афиши (2024 г.).
- **Фильтрация:** оставлены только заказы с `mobile`/`desktop`, исключён тип «фильм».
- **Валюта:** выручка приведена к RUB с учётом динамики курса тенге.
- **Очистка:** удалены дубликаты, выбросы по `revenue_rub` (99-й перцентиль), отрицательные значения, оптимизированы типы данных.
- **Итоговая выборка:** 21 854 уникальных пользователя с сформированными профилями.

## Ключевые выводы
1. **Потенциал удержания:** ~60% аудитории совершили 2+ заказа, что указывает на высокий потенциал retention-стратегий.
2. **Оптимальный профиль лояльного клиента:** 
   - Первый заказ на концерты или театр.
   - Средний чек 400–600 ₽, 2–3 билета в заказе.
   - Интервал между покупками ~10 дней.
3. **Влияние признаков:**
   - `avg_tickets` и `avg_revenue` имеют слабую положительную связь с общим количеством заказов.
   - `avg_days_since_prev` имеет слабую отрицательную связь (чем короче интервал, тем выше лояльность).
   - День недели первого заказа и тип устройства **не оказывают** статистически значимого влияния на возвратность.
4. **Проверка гипотез:** 
   - Тип мероприятия влияет на возврат (частично подтверждено: концерты/театры показывают лучшие метрики).
   - Регион активности не является сильным предиктором удержания (гипотеза не подтверждена).

## Рекомендации для бизнеса
-  **Персонализация:** таргетировать коммуникации на сегмент с чеком 400–600 ₽ и интересом к live-мероприятиям.
-  **Триггерные рассылки:** настроить автоматические уведомления через 7–10 дней после последнего заказа.
-  **Работа с новым трафиком:** внедрить приветственные скидки для пользователей с первым чеком <200 ₽.
-  **Мониторинг аномалий:** отслеживать экстремально высокую активность (возможные боты/корп. закупки) для очистки аналитических выборок.
